Detailed Description

The Farmer class represents a farmer in the game.

Farmer has following production-efficiency:

  • Money - 0.25
  • Food - 5.00
  • Wood - 0.75
  • Stone - 0.50
    Farmer consumes food, money, wood and a little bit of stone.
    Resourcefoucs on food.
