Detailed Description

The Lumberjack class represents a lumberjack in the game.

Lumberjack has following production-efficiency:

  • Money - 0.25
  • Food - 5.00
  • Wood - 0.75
  • Stone - 5.00
    Lumberjack consumes money, food, a lot of wood and stone.
    Resourcefoucs on wood.
