Detailed Description

The Miner class represents a miner in the game.

Miner has following production-efficiency:

  • Money - 0.25
  • Food - 1.00
  • Wood - 0.75
  • Stone - 5.00
    Miner consumes money, food, wood and a lot of stone.
    Resourcefoucs on stone.
