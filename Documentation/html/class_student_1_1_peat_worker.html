Detailed Description

The Peatworker class represents a "swamp worker" in the game.

Peatworker has following production-efficiency:

  • Money - 5.00
  • Food - 1.00
  • Wood - 0.75
  • Stone - 0.50
    Peatworker consumes money, food and wood.
    Resourcefoucs on money.
